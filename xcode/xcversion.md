# xcode-install

> 安装不同版本Xcode

[Github](https://github.com/neonichu/xcode-install)

## 安装 xcode-install

```shell
$ gem install xcode-install
```

## 查看可安装的Xcode版本

```shell
$ xcversion list
```

## 安装 Xcode

```shell
$ xcversion install 7
```

## 安装模拟器

查看已安装模拟器

```sh
xcversion simulators
```
