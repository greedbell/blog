# adb 命令

* [ADB Commands](https://adbshell.com/)

## 安装 apk

```sh
# 查看设备列表
adb devices
# 安装 apk
adb install <apk path>
```

## shell

```sh
# 获取 root 权限，否则 adb shell 后运行任何命令提示 “Permission denied”
adb root
# 查看设备列表
adb devices
# 进入设备 shell
adb -s emulator-5554 shell
```
